# Jeremy Lockhart – Interactive Portfolio

This is my personal portfolio page built with HTML, CSS, and JavaScript.

The goal of this project was to practice working with JavaScript objects, session storage, and dynamic DOM updates while keeping everything clean and simple.

---

## 🚀 What This Page Does

- Stores project information using custom JavaScript objects  
- Saves project data using `sessionStorage` and `JSON.stringify()`  
- Loads saved data using `JSON.parse()`  
- Dynamically builds project cards on page load  
- Includes a contact form with a message area  
- Simulates sending a message with a short delay  
- Includes a dark mode toggle (saved using localStorage)  
- Shows a simple welcome modal when the page loads  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS – no external libraries)

---

## 📦 Dependencies

There are no external libraries required.  
Everything runs in a modern web browser like:

- Chrome  
- Edge  
- Firefox  

---

## ▶️ How to Run the Project

1. Download the files.
2. Open `index.html` in your browser.
3. That’s it!

Note:  
Project data is saved in **sessionStorage**, so it will reset if you close the browser tab.

This project helped me practice:
